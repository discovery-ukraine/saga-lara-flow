# Security Policy

## Supported versions

The latest released `1.x` line receives security fixes.

## Reporting a vulnerability

If you discover a security vulnerability within `saga-lara-flow`, please report it privately rather
than opening a public issue.

- Use GitHub's [private vulnerability reporting](https://github.com/discovery-ukraine/saga-lara-flow/security/advisories/new), or
- Email **mkrnnk@gmail.com** with a description and reproduction steps.

Please do **not** disclose the issue publicly until a fix has been released. All reports will be
reviewed and addressed promptly, and you will be credited unless you prefer to remain anonymous.
